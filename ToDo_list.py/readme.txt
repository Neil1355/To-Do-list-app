To-Do List (Tkinter)

This is a simple To-Do List desktop app made using Python and Tkinter.

Features:
- Add new tasks
- Check off completed tasks
- Delete completed tasks
- Double-click to edit any task
- Automatically saves tasks to a file (tasks.txt)
- Loads saved tasks when reopened

How to Run:
1. Make sure you have Python 3 installed
2. Save the script as todo_app.py
3. Run it using: python todo_app.py

No external libraries are needed.
